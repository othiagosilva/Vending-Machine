Este projeto foi desenvolvido durante o 1º semestre do curso de Análise e Desenvolvimento de Sistemas, utilizando
a linguagem C++. O software visa simular uma operação de compra em uma máquina de vendas.

Ao iniciar o programa é exibido sua interface com os produtos à venda e uma pergunta se deseja iniciar a compra ou não - interação adicionada visando uma maior interatividade, como não há visualização do produto o usuário pode querer ver o que está a venda e não querer comprar nada -. Entrando no modo de compra é solicitado o número do item e o valor a ser colocado na máquina. Com todos os passos realizados a máquina devolve o troco e pergunta se quer comprar algo mais.
